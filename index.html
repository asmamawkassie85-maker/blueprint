1. Development Categories

Software development has many categories, but as a beginner, you should focus on these two main areas:

Web Development

Creating websites and web applications that run in browsers. This includes everything from simple blogs to complex web apps like Facebook or Gmail.

  • Frontend: What users see and interact with
  • Backend: Server, database, and application logic
  • Full-Stack: Both frontend and backend

Mobile Development

Creating applications for smartphones and tablets. You can build native apps (specific to iOS or Android) or cross-platform apps (work on both).

  • Native iOS: Swift, Objective-C
  • Native Android: Kotlin, Java
  • Cross-Platform: React Native, Flutter

My Recommendation: Start with Web Development first, then move to mobile. The concepts you learn will transfer easily, and React Native will allow you to build mobile apps with your web knowledge.

2. How Everything Works Together

Modern applications are built using a layered architecture. Here's how all the pieces fit together:

Frontend

HTML, CSS, JavaScript - what users see and interact with

APIs

Communication bridge between frontend and backend

Backend

Server, database, and application logic

Database

Stores and manages application data

Key Technologies & Frameworks

Frontend(web)

React(u gonna focus on this), Vue, Angular

Mobile

React Native Flutter

Backend(the same for web and mobile)

Node.js (Express), Python (Django, FastAPI), Rust

APIs(don't worry about this

REST, GraphQL, RESTful APIs
